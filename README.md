### Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Web browser (Chrome, Firefox, Safari, Edge)

### Step-by-Step Guide

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/delwaththa-farm.git

# 2. Navigate to project directory
cd delwaththa-farm

# 3. Create database
mysql -u root -p
CREATE DATABASE delwaththa_farm;
exit;

# 4. Import database schema
mysql -u root -p delwaththa_farm < database/schema.sql

# 5. Configure database connection
# Edit config/database.php with your credentials

# 6. Set proper permissions
chmod -R 755 assets/
chmod -R 755 uploads/

# 7. Start your web server
# For Apache: Place in htdocs/www folder
# For PHP built-in server:
php -S localhost:8000
```

### Database Configuration

Edit `config/database.php`:

```php
<?php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', 'your_password');
define('DB_NAME', 'delwaththa_farm');

// Create connection
$conn = mysqli_connect(DB_HOST, DB_USER, DB_PASS, DB_NAME);

// Check connection
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
?>
```
